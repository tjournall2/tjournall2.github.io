TJournal ищет Android-разработчика

Ищем второго Android-разработчика для Android-разработки приложений TJ, vc.ru и DTF. Можно работать как удалённо из любой точки мира, так и в наших офисах в Санкт-Петербурге или в Москве.

5000 просмотров

Текущее состояние проекта:

  • minSDK 19;
  • 30% Java и 70% Kotlin (постепенно переписываем приложения на второй);
  • Product Flavors и Build Variants;
  • Retrofit, Picasso, RxJava2;
  • Jetpack, Android X;
  • SquiDB;
  • Butterknife (активно от него избавляемся).

Храним всё это добро на Github, планируем задачи в Jira, загружаем макеты в Zeplin и Figma, а общаемся друг с другом через Slack.

Если всё вышеперечисленное для вас знакомо, то в качестве тестового задания предлагаем создание приложения для написания постов в подсайт TJgram и отображения в виде Инстаграм-ленты. В качестве примера дизайна можно взять текущее приложение Инстаграма.

Подсайт TJgram создан для авторских снимков и видео пользователей TJ.

Что нужно реализовать:

  • Авторизация через QR код;
  • Создание записи и прикрепление к нему изображения;
  • Отображение ленты записей только с прикрепленным изображением в виде ленты Инстаграма;
  • Лайк записи.

Если вышеперечисленного мало: поддержку воспроизведения MP4 видео, автообновление рейтинга записей с помощью Websocket (есть в документации) и кэширование записей для показа оффлайн.

Для доступа к документации и подсайту Git необходимо заполнить форму.

Ждём ваши письма с выполненными заданиями в виде проектов на Github, пару строк о себе со ссылками на свои профили в соцсетях, ожидания по заработной плате, а также ответ на вопрос: «Почему вы хотите работать с нами?» по адресу [email protected].
